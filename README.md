College Admission Database ProjectProject OverviewThis project is a complete College Admission & Student Management Database System developed using MySQL. It demonstrates professional database design, strict data normalization, cross-table relationships, constraints, and operational business intelligence metrics used in a real-world higher education platform.The database comprehensively manages academic departments, degree programs, course catalogs, term schedules, faculty workloads, applicants, document verification pipelines, active student records, section allocations, grade books, tuition billing, financial aid, and security audit logs.FeaturesCore Academic FrameworkAutomate academic department and physical building mappings.Track multi-tier programmatic degree paths (Undergraduate, Postgraduate, PhD).Manage university course catalogs with strict unique code criteria.Structure dynamic timeline frameworks for active and staging terms.Admission Pipeline & VettingRegister and maintain extensive applicant contact and demographic profiles.Run automated screening pipelines based on high school GPAs.Track multi-state applications (Submitted, Under Review, Accepted, etc.).Implement faculty-vetted mandatory file and transcript verification checkpoints.Active Enrollment & SchedulingPromote verified applicants into active students with structured university roll numbers.Configure physical room seat capacities and block overfill tracking.Route many-to-many bridges mapping student course enrollments.Track instructor distributions and advisor student workloads.Logistics, Billing & Security AuditingProcess tuition invoices dynamically via an expression-based Generated Column.Apply institutional financial aid and scholarship percentage offsets.Map on-campus residential dormitory space management and expected rent yields.Record structural system data changes onto administrative security audit logs.Database SchemaThe
